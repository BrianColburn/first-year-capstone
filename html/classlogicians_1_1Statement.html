Detailed Description

A class that represents boolean expressions. Statement provides functionality for storing, manipulating, and displaying boolean expressions.

◆ collect_expressions()

std::vector< Statement > logicians::Statement::collect_expressions ( ) const

Collect all the expressions of a statement.

Returns
a vector containing each sub-expression and the top-level the statement. De-duplication is not performed.
Example
parse_string("((a)^(b))v(~(a))").collect_expressions() ->
{"a", "b", "(a)^(b)", "a", "~(a)", "((a)^(b))v(~(a))"}

◆ collect_expressions_ordered()

std::vector< Statement > logicians::Statement::collect_expressions_ordered ( ) const

Collect all the expressions of a statement.

Returns
a vector containing each sub-expression and the top-level the statement. It is guaranteed that variables come before all other expressions and each element is unique.
Example
parse_string("((a)^(b))v(~(a))").collect_expressions_ordered() ->
{"a", "b", "(a)^(b)", "~(a)", "((a)^(b))v(~(a))"}
